## Setup Instructions

1. **Add Your Profile Image**
   - Replace `profile.jpg` with your actual photo
   - Recommended size: 500x500 pixels (square aspect ratio)
   - Supported formats: JPG, PNG, WebP

2. **Update Project Images** (Optional)
   - Add images for your projects in the Projects section
   - Update the project cards with your actual work

3. **Run Locally**
   - Simply open `index.html` in your web browser
   - No build process required!

## Customization

### Change Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #3b82f6;  /* Main blue color */
    --secondary-color: #8b5cf6; /* Purple accent */
    --accent-color: #06b6d4;    /* Cyan accent */
}
```

### Update Typing Animation
Modify the phrases array in `script.js`:
```javascript
const phrases = [
    'Your Phrase 1',
    'Your Phrase 2',
    // Add more phrases...
];
```

### Add More Projects
Add project cards in the projects section of `index.html`.
